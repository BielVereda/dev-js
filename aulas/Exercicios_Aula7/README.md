### **Atividade Prática**

1. Crie dois arquivos: index.html e app.js.
2. Implemente:
    - Um array com **5 frutas**. Use push, pop, unshift e shift para alterar a lista.
    - Um array de **10 números**. Mostre apenas os pares usando filter.
    - Use map para criar um novo array com os números ao quadrado.
    - Use reduce para somar todos os números.
    - Use sort para ordenar e reverse para inverter.
3. Mostre os resultados no navegador e no terminal.
