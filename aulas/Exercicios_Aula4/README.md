### **Atividade Prática**

1. Crie dois arquivos: `index.html` e `app.js`.

2. No **HTML (navegador):**
    - Pergunte ao usuário com `prompt` a idade.
    - Mostre se é **criança (<12)**, **adolescente (12-17)**, **adulto (18-59)** ou **idoso (60+)**.

3. No **Node.js (terminal):**
    - Crie uma variável `mes` (1 a 12).
    - Use `switch` para mostrar o nome do mês.

4. Extra: crie um programa que receba uma nota e exiba:
    - `>= 9`: Excelente
    - `>= 7`: Bom
    - `>= 5`: Regular
    - `< 5`: Reprovado
