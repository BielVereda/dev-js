### **Atividade Prática**

1. Crie dois arquivos:
    - `index.html` (para rodar no navegador).
    - `app.js` (para rodar no Node.js).

2. Em cada um, exiba:
    - Seu nome.
    - Sua idade.
    - O resultado da soma entre `7 + 3`.
    
3. Teste rodar:
    - `index.html` → abra no navegador.
    - `app.js` → rode com `node app.js` no terminal.
